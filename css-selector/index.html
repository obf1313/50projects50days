• 简单选择器

    • *

      选择所有元素。

      你看所有元素的背景都变成了灰色

    • element

      例子：li，选择所有 li 元素。

      现在所有 li 元素没有 . 了，变成数字了

    • .class

      选择 class="intro" 的所有元素。

      我的 class 为 intro

    • #id

      选择 id="first-name" 的元素。

      我的 id 是 first-name

    • .class1.class2

      选择 class 属性中同时有 name1 和 name2 的所有元素。

      我同时有 class1 和 class2

    • element.class

      选择 class="intro" 的所有 p 元素。

      我是 class="intro" 的 p 元素

    • element,element

      选择所有 div 元素和所有 p 元素

      我是 div

      我是 p

  • 组合器选择器

    • .class1 .class2（后代选择器）

      选择作为类名 class1 元素后代的所有类名 class2 元素。

      我是 class1 的后代 class2

    • element element（后代选择器）

      选择 div 元素内的所有 p 元素

      我是 div 内的 p 元素

    • element>element（子选择器）

      选择父元素是 div 的所有 p 元素。

      我是 p，我的父元素是 div

    • element+element（相邻兄弟选择器）

      选择父元素是 div 的所有 p 元素。

      我是 p，我的父元素是 div

    • element1~element2（通用兄弟选择器）

      选择紧跟 div 元素的首个 p 元素。

      我是 div 元素

      我是 p1 元素

      我是 p2 元素

  • 属性选择器

    • [attribute]

      选择带有 target 属性的所有元素。

      我是 a 元素，我有 target 属性

    • [attribute=value]

      选择带有 target="_blank" 属性的所有元素。

      我是 a 元素，我 target="_blank"

    • [attribute~=value]

      选择 title 属性包含单词 "flower" 的所有元素。（需要是单词，空格区分那种）

      我的 title 中包含 flower
      我的 title 中不包含 flower

    • [attribute|=value]

      选择 lang 属性值以 "en" 开头的所有元素。

      我的 lang 属性以 "en" 开头。

    • [attribute^=value]

      选择其 href 属性值以 "https" 开头的每个 a 元素。

      我的 href 属性为 https 开头

    • [attribute$=value]

      选择其 href 属性以 ".pdf" 结尾的所有 a 元素。

      我的 href 属性为 .pdf 结尾

    • [attribute*=value]

      选择其 href 属性值中包含 "abc" 子串的每个 a 元素。（测试貌似这个优先级最高）

      我的 href 属性包含 "aaa" 字符串

  • 伪类选择器

    • :active

      选择活动链接。

      我的 active 样式颜色变成了红色

    • :checked（我不理解）

      选择每个被选中的 input 元素。

    • :default

      选择默认的 input 元素。

    • :disabled

      选择每个被禁用的 input 元素。

    • :empty

      选择没有子元素的每个 p 元素（包括文本节点）。

    • :enabled

      选择每个启用的 input 元素。

    • :first-child

      选择属于父元素的第一个子元素的每个 p 元素。

      first-child

      last-child

    • :last-child

      选择属于其父元素最后一个子元素每个 p 元素。

      first-child

      last-child

    • :first-of-type

      选择属于其父元素的首个 p 元素的每个 p 元素。

      谜语人，看不懂

    • :last-of-type

      选择属于其父元素的最后 p 元素的每个 p 元素。

      谜语人，看不懂

    • :focus

      选择获得焦点的 input 元素。

    • ：fullscreen

      选择处于全屏模式的元素。

    • :hover

      选择鼠标指针位于其上的链接。

      鼠标放上去变粉色

    • :in-range

      选择其值在指定范围内的 input 元素。

    • out-of-range

      选择值超出指定范围的 input 元素。

    • :indeterminate

      选择处于不确定状态的 input 元素。

    • :invalid

      选择具有无效值的所有 input 元素。

    • :valid

      选择带有有效值的所有 input 元素。

    • :lang(language)

      选择 lang 属性等于 "it"（意大利）的每个 p 元素。

      意大利

      中国

    • :link

      选择所有未访问过的链接。

      我没有被点击过

    • :not(selector)

      选择非 p 元素的每个元素。

      我不是 p 元素

      我不是 p 元素

      我是 p 元素

    • :nth-child(n)

      选择属于其父元素的第二个子元素的每个 p 元素。

      第一个 p 元素

      第二个 p 元素

    • nth-last-child(n)

      选择属于其父元素的第二个子元素的每个 p 元素，从最后一个子元素开始计数。

      第一个 p 元素

      第二个 p 元素

    • nth-of-type(n)

      选择属于其父元素第二个 p 元素的每个 p 元素。

    • nth-last-of-type(n)

      同上，但是从最后一个子元素开始计数。

    • :only-of-type

      选择属于其父元素唯一的 p 元素的每个 p 元素。

    • :only-child

      选择属于其父元素的唯一子元素的每个 p 元素。

    • :optional

      选择不带 "required" 属性的 input 元素。

    • :required

      选择已规定 "required" 属性的 input 元素。

    • :read-only

      选择已规定 "readonly" 属性的 input 元素。

    • :read-write

      选择未规定 "readonly" 属性的 input 元素。

    • :root

      选择文档的根元素。

    • :target

      选择当前活动的 #news 元素。锚点相关

      target1 target2
      target1
      target2

    • :visited

      选择所有已访问的链接。

      bing
