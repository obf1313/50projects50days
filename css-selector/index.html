• .class

    选择 class="intro" 的所有元素。

    我的 class 为 intro

  • .class1.class2

    选择 class 属性中同时有 name1 和 name2 的所有元素。

    我同时有 class1 和 class2

  • .class1 .class2

    选择作为类名 class1 元素后代的所有类名 class2 元素。

    我是 class1 的后代 class2

  • #id

    选择 id="first-name" 的元素。

    我的 id 是 first-name

  • *

    选择所有元素。

    你看所有元素的背景都变成了灰色

  • element

    例子：li，选择所有 li 元素。

    现在所有 li 元素没有 . 了

  • element.class

    选择 class="intro" 的所有 p 元素。

    我是 class="intro" 的 p 元素

  • element,element

    选择所有 div 元素和所有 p 元素

    我是 div

    我是 p

  • element element

    选择 div 元素内的所有 p 元素

    我是 div 内的 p 元素

  • element>element

    选择父元素是 div 的所有 p 元素。

    我是 p，我的父元素是 div

  • element1~element2

    p~ul 选择前面有 p 元素的每个 ul 元素。

    我是 p 元素

      我是 ul 元素 1
      我是 ul 元素 2

  • [attribute]

    选择带有 target 属性的所有元素。

    我是 a 元素，我有 target 属性

  • [attribute=value]

    选择带有 target="_blank" 属性的所有元素。

    我是 a 元素，我 target="_blank"

  • [attribute~=value]

    选择 title 属性包含单词 "flower" 的所有元素。（需要是单词，空格区分那种）

    我的 title 中包含 flower
    我的 title 中不包含 flower

  • [attribute|=value]

    选择 lang 属性值以 "en" 开头的所有元素。

    我的 lang 属性以 "en" 开头。

  • [attribute^=value]

    选择其 href 属性值以 "https" 开头的每个 a 元素。

    我的 href 属性为 https 开头

  • [attribute$=value]

    选择其 href 属性以 ".pdf" 结尾的所有 a 元素。

    我的 href 属性为 .pdf 结尾

  • [attribute*=value]

    选择其 href 属性值中包含 "abc" 子串的每个 a 元素。（测试貌似这个优先级最高）

    我的 href 属性包含 "aaa" 字符串
